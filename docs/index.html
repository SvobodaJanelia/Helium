Helium

The open source DataJoint companion

GitHub License

Works with any MySQL database

Helium was built to work with any MySQL database, but to be aware of DataJoint conventions. As a result, Helium is smarter than your average database companion and can understands concepts like the difference between a manual and a computed table.

Makes data entry easy

Helium is capable of building complex forms for any table and even handles master/part table relationships.

RESTful API

Every bit of data availble when using Helium is also available programmatically—from schema names to table content and every bit of metadata in between.

Open source

Helium is available for free (as in speech) and its source code is available on GitHub under the MIT license.
